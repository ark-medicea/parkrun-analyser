➕ Add an Athlete

To add a new athlete to the dashboard, you'll need their parkrun athlete ID. This opens a request on GitHub to add them.

How to find an athlete ID

  1. Go to Cassiobury parkrun results
  2. Find the person in the results list
  3. Click their name — their profile URL will be like:
    parkrun.org.uk/parkrunner/1234567/
  4. The number in the URL is their athlete ID

⚙️ Direct Config Edit

If you have write access to the repository, you can also edit the config directly:

Edit config.json on GitHub →

Add the athlete's surname to the surnames array. The scraper will discover them automatically on the next run.
